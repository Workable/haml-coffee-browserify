Browserify plugin to convert haml-coffee (.hamlc) templates to Javascript.

# Installation

This package is available on npm as:

```
npm install haml-coffee-browserify
```

# Usage

```
$ browserify -t haml-coffee-browserify foo.hamlc > bundle.js
```
